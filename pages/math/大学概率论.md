- 第4讲 随机变量的数字特征
	- 数学期望
		- 随机变量
			- 离散型
			- 连续型
		- 随机变量的函数
		- 性质
			- E(C) = C
			- E(CX) = CE(X)
			- E(X+Y) = E(X) + E(Y)
			- 若X,Y相互独立，E(XY) = E(X)E(Y)
	- 方差
		- D(X) = E [X - E(X)]^2
			- = E(X^2) - [E(X)]^2
		- 性质
			- D(C) = 0
			- D(CX) = C^2 * D(X) = D(CX + B)
			- 若相互独立， D(X+Y) = D(X) + D(Y)
			- 方差为0的充分必要条件
		- 标准差（均方差）
		- 标准化随机变量
	- 常见分布的数学期望and方差
	- 协方差、相关系数与矩
